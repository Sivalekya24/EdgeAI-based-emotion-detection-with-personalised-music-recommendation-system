Convolutional Neural Network (CNN)

CNNs are specialised neural networks designed for processing visual data. They automatically learn hierarchical patterns in images — edges, textures, expressions — making them ideal for emotion detection from facial images.

🔍

Convolutional Layers

Extract features like edges, textures, and expression patterns from facial images using learnable filters at multiple scales

📉

Pooling Layers

Reduce spatial dimensions while retaining the most important feature information, improving computational efficiency

🔗

Dense Layers

Combine all learned features to make the final emotion classification decision across all 7 emotion categories

Training Dataset & Emotion Classes

The CNN model is trained on a large-scale facial expression dataset — tens of thousands of labelled images preprocessed to 48×48 grayscale. The model classifies 7 emotion categories:

😊 Happy 😢 Sad 😠 Angry 😨 Fear 🤢 Disgust 😲 Surprise 😐 Neutral

Image Processing Pipeline

Every captured face image goes through a rigorous multi-stage validation and processing pipeline before reaching the CNN — ensuring only clean, usable frames are analysed.

📷

1. Image Acquisition

User captures face via browser webcam → Image encoded as base64 JPEG → Sent to Flask server → Validated for format and size constraints

🔧

2. Preprocessing

Convert to grayscale → Resize to 48×48 pixels → Normalise pixel values (0–1 range) → Brightness check against minimum threshold (MIN_BRIGHTNESS=60)

✅

3. Validation Gates

Face detection check → Face area must be within bounds (8,000–120,000 px²) → Alignment tolerance checked → Head tilt stability verified via MediaPipe facial landmarks

🧠

4. Model Inference

Feed preprocessed image into CNN → Forward propagation through convolutional layers → Extract 7-class probability vector via softmax activation

📊

5. Post-processing

Apply softmax to get probabilities → Select highest confidence prediction → Map to emotion label → Confidence must exceed threshold (CONFIDENCE_THRESHOLD=0.45)

🎵

6. Result Generation

Emotion label + all 7 probabilities returned to browser → User selects language → Spotify engine maps emotion to keywords → Real tracks fetched and displayed
